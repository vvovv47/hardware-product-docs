## II. 核心控制功能

### 2.1 移动控制

#### 2.1.1 差速驱动运动学

石头 G10S Pro 采用双轮差速驱动方式，通过控制左右轮的转速差实现前进、后退和转向。

**运动学方程：**

```
图2-1 差速驱动运动学模型

                    前进方向
                        ↑
                        │
           ┌────────────┼────────────┐
           │            │            │
           │    ┌───────┴───────┐    │
           │    │               │    │
      V_L ←────┤      ●        ├────→ V_R
      ω_L      │    中心       │      ω_R
           │    │               │    │
           │    └───────┬───────┘    │
           │            │            │
           │←──── L ────→│            │
           └────────────┼────────────┘
                        │
                        
运动学方程：
  v = (V_R + V_L) / 2          线速度
  ω = (V_R - V_L) / L          角速度
  
逆运动学方程：
  V_R = v + ω × L / 2          右轮速度
  V_L = v - ω × L / 2          左轮速度
  
里程计计算：
  Δx = v × cos(θ) × Δt
  Δy = v × sin(θ) × Δt
  Δθ = ω × Δt
```

### 2.3 运动规划

#### 2.3.1 全局路径规划

石头 G10S Pro 采用改进的A*算法进行全局路径规划，结合清扫覆盖需求生成最优清扫路径。

**全局规划算法流程：**

```mermaid
flowchart TD
    A[接收清扫任务] --> B[加载地图数据]
    B --> C[确定起点终点]
    C --> D[生成覆盖区域]
    D --> E{区域类型判断}
    
    E -->|全屋清扫| F[生成弓字形路径]
    E -->|选区清扫| G[生成区域路径]
    E -->|划区清扫| H[生成指定路径]
    
    F --> I[路径优化]
    G --> I
    H --> I
    
    I --> J[路径平滑处理]
    J --> K[生成轨迹点序列]
    K --> L[输出全局路径]
```

#### 2.3.2 覆盖路径规划

### 3.1 动力学建模

#### 3.1.1 机器人动力学模型

石头 G10S Pro 的动力学模型考虑了质量、惯量、摩擦力等因素。

**动力学方程：**

```
图3-1 机器人动力学模型

                    ┌─────────────────────┐
                    │                     │
                    │    机器人本体       │
                    │    质量: m          │
                    │    惯量: I          │
                    │                     │
                    └──────────┬──────────┘
                               │
              ┌────────────────┼────────────────┐
              │                │                │
         ┌────┴────┐     ┌────┴────┐     ┌────┴────┐
         │ 左轮    │     │ 中心    │     │ 右轮    │
         │ τ_L     │     │         │     │ τ_R     │
         │ F_L     │     │         │     │ F_R     │
         └─────────┘     └─────────┘     └─────────┘

运动方程：
  m × dv/dt = F_L + F_R - F_resistance
  I × dω/dt = (F_R - F_L) × L/2 - τ_resistance

其中：
  F_L, F_R: 左右轮驱动力
  τ_L, τ_R: 左右轮扭矩
  F_resistance: 运动阻力（摩擦力、坡度阻力等）
  τ_resistance: 转向阻力矩
```
